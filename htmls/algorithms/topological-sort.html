拓扑排序 Topological Sort

每次取入度为0的点输出，删掉它的出边。全部输出 → 无环。

核心思想

想象大学选课：有些课有先修要求。拓扑排序就是找一个上课顺序，保证每门课的先修课都排在它前面。

做法：找到没有先修课（入度=0）的课先上，上完后把它从图中"删掉"（它后继的入度减1）。重复直到所有课都排完，或者发现排不完（有环）。

Kahn/BFS 法模板

from collections import deque

def topo_sort(n, edges):
    g = [[] for _ in range(n)]
    indeg = [0] * n
    for u, v in edges:
        g[u].append(v)
        indeg[v] += 1
    q = deque(i for i in range(n) if indeg[i] == 0)
    order = []
    while q:
        u = q.popleft()
        order.append(u)
        for v in g[u]:
            indeg[v] -= 1
            if indeg[v] == 0:
                q.append(v)
    return order if len(order) == n else []  # 空 = 有环

DFS 法（三色标记）

白色=未访问，灰色=正在访问（在递归栈中），黑色=已完成。

灰色碰灰色就是环！完成顺序的逆序就是拓扑序。

应用

  • 课程表：判断能否修完所有课（LC 207/210）
  • 字典序最小拓扑序：把队列换成最小堆
  • DAG 上 DP：按拓扑序处理，每个节点的 DP 值只依赖已算好的前驱

思考题

BFS 法和 DFS 法各有什么优势？

BFS (Kahn) 直观、容易理解，天然产生拓扑序，方便改成"字典序最小"版本（用堆）。DFS 法代码更短，适合判环（灰碰灰），且后序逆序天然是拓扑序，适合需要逆拓扑序的场景（如 SCC）。

如果要求字典序最小的拓扑序怎么做？

把 BFS 的普通队列换成最小堆（优先队列）。每次从入度为0的节点中取编号最小的。注意这和"最短路径"无关，只是输出顺序的要求。

拓扑排序和 DAG 上的 DP 有什么关系？

DAG 上 DP 的转移顺序就是拓扑序：当你处理节点 u 时，它的所有前驱都已经被处理完了，所以 dp[u] 可以从前驱的 dp 值转移过来。经典例子：DAG 最长路径（LC 329）。
