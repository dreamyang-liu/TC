DP Attention: 让 KV Cache 不再复制

对比传统 TP+EP 方案 vs DP Attention + EP 方案 (4 GPU, 8 Experts 示例)

传统方案: TP (Attention) + EP (Experts)
DP Attention 方案: DP (Attention) + EP (Experts)
对比总结

❌ 传统 TP + EP

Attention 通信AllReduce (每层 2×)
Expert 通信All-to-All (每层 2×)
KV Cache 副本数EP/TP 份 (冗余!)
KV Cache 总占用× EP/TP 倍
Attention 参数按 TP 切分

✓ DP Attention + EP

Attention 通信零! (各算各的)
Expert 通信All-to-All (每层 2×)
KV Cache 副本数1 份 (零冗余)
KV Cache 总占用最小化
Attention 参数每卡复制 (~1% 参数)
