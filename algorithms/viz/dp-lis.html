📈 DP · 最长递增子序列 (LIS)

nums = [10, 9, 2, 5, 3, 7, 101, 18]

核心 AHA：dp[i] = "以 nums[i] 结尾的 LIS 长度"。 要算 dp[i]，看前面所有 j（j<i 且 nums[j]<nums[i]）的 dp[j]，取最大 + 1。 没有这种 j 就是 1（自己一个数）。
最终答案 = max(dp)，不是 dp[n-1]。

def lis(nums):
    n = len(nums)
    dp = [1] * n              # 每个数自己至少能凑出长度 1
    for i in range(n):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)
