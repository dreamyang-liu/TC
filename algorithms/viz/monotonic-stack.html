"下一个更大元素"：数组 [2, 1, 5, 3, 4]，找每个数右边第一个比它大的

核心 AHA：栈里存的是"还在等下一个更大值"的元素。 新元素来了，比它们都大 → 它就是它们等的那个！所有"被它打败"的人一次性出栈。 每个元素 最多入栈出栈各一次 → 总 O(n)。

def next_greater(nums):
    n = len(nums); ans = [-1]*n; stack = []
    for i, x in enumerate(nums):
        while stack and nums[stack[-1]] < x:
            ans[stack.pop()] = x        # x 是栈顶的下一个更大
        stack.append(i)
    return ans
