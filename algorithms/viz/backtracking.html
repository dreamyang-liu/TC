求 [1,2,3] 的所有子集

核心 AHA：回溯 = "选 → 递归 → 撤销" 三件套。 每个节点的所有 children 共享同一个 path 变量，做完一个子分支必须把状态恢复成原样， 才能去走下一个。这就是回溯的"撤销"。

def subsets(nums):
    ans = []
    def dfs(start, path):
        ans.append(path[:])           # 1) 收集
        for i in range(start, len(nums)):
            path.append(nums[i])      # 2) 选
            dfs(i + 1, path)         # 3) 递归
            path.pop()                # 4) 撤销 ← 关键！
    dfs(0, [])
    return ans
